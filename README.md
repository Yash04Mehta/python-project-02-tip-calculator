# Tip Calculator 💰

This is a beginner Python project that calculates how much each person should pay when splitting a bill including a tip.

## How it works

The program:
- asks the total bill amount
- asks the tip percentage (10%, 12%, or 15%)
- asks the number of people splitting the bill
- calculates how much each person should pay

## Example

Input:
Bill: 1500
Tip: 10%
People: 3

Output:
Each person should pay: 550.0

- variables
- user input
- type conversion
- arithmetic operations
- formatted output
